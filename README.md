# React GST Invoice Generator

A React-based GST Invoice Generator and Calculator with **light/dark mode**, dynamic calculations, and PDF export functionality. Built for small businesses or individuals who need quick, accurate GST invoices with a modern, responsive interface.

---

## **Features**

- **Profile Section:** Displays user name, image, about info, and interests.
- **Light/Dark Mode:** Toggle between light and dark themes using Tailwind CSS.
- **GST Calculator & Invoice Generator:**
  - Dynamically calculates subtotal, GST (CGST + SGST), and total amount.
  - Inputs for item name, quantity, and price.
  - Real-time invoice display in a modal.
- **PDF Export:** Generates a professional invoice in PDF format using `jsPDF`.
- **Responsive Design:** Works on both desktop and mobile devices.
- **Modern UI:** Built with React, Tailwind CSS, and React Icons.

---

## **Technologies Used**

- **React.js** – Frontend framework  
- **Tailwind CSS** – Styling and dark mode  
- **jsPDF & jspdf-autotable** – PDF generation  
- **React Icons** – Social and UI icons
